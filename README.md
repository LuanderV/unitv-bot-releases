# UniTV Bot — Releases

Distribuição pública do **UniTV Bot** (app desktop). Aqui ficam só os pacotes de
instalação/atualização — o código-fonte é privado.

## Baixar

Versão mais recente em **[Releases](../../releases/latest)** → `UniTV-Bot-latest.zip`.

O download oficial é pelo painel (aba **Bot WhatsApp → Download**); o app também
se atualiza sozinho quando sai uma versão nova.

## Arquivos de cada release

- `UniTV-Bot-latest.zip` — nome estável (é o que o painel e o auto-update baixam)
- `UniTV-Bot-vX.Y.Z.zip` — cópia versionada (histórico)
- `latest.json` — `{ version, sha256, zip }` (integridade do auto-update)

Publicado automaticamente pelo CI do repositório privado a cada tag `v*`.
